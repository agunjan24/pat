# PAT — Portfolio Analytic Tool

A tool to visualize, analyze, and track investment portfolios with algorithmic signal generation for risk-adjusted buy/sell decisions across stocks, options, and LEAPS.

## Overview

PAT provides:

- **Portfolio Tracking** — Aggregate and monitor holdings across asset classes (stocks, options, LEAPS)
- **Visualization** — Interactive charts and dashboards for portfolio performance, allocation, and risk metrics
- **Analytics** — Quantitative analysis of portfolio characteristics including returns, volatility, correlation, and drawdowns
- **Signal Generation** — Algorithms to identify optimal entry/exit points using risk-adjusted models

## Core Modules

| Module | Purpose |
|--------|---------|
| `tracker` | Portfolio ingestion, position management, and P&L tracking |
| `visualizer` | Charts, dashboards, and interactive visualizations |
| `analyzer` | Risk metrics, performance attribution, and statistical analysis |
| `signals` | Buy/sell signal algorithms with risk-adjusted scoring |

## Asset Coverage

- **Equities** — Individual stocks and ETFs
- **Options** — Calls, puts, spreads, and multi-leg strategies
- **LEAPS** — Long-term equity anticipation securities with theta/decay modeling
